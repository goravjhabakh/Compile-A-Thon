Make sure you have clang + llvm
https://github.com/llvm/llvm-project/releases/tag/llvmorg-18.1.8
Download tar file and extract it
Add the bin folder path to PATH in system variables

Install streamlit for frontend
pip install streamlit

To run the app
streamlit run file_path

Easiest way to run is to open terminal in this folder and do streamlit run main.py
